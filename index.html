Projects

Skoolie Quest

Skoolie Quest is an all-in-one travel and community platform built for school bus conversion enthusiasts and nomadic adventurers. Designed to simplify life on the road, the site features a custom GPS navigation tool to locate campgrounds, build resources, and essential stops, all tailored for skoolie travelers. Users can connect through a built-in social platform to share tips, stories, and build ideas. With a vibrant and functional design, Skoolie Quest balances exploration with practicality—empowering users to plan, connect, and thrive on their journey, one stop at a time.

Weather Messages

Weather Messages is a simple yet engaging JavaScript program that generates a random weather-related message each time it runs. Designed to add a bit of fun and unpredictability, the program randomly selects from a collection of quirky, motivational, or weather-themed phrases—no live data or user input required. It’s a great demonstration of JavaScript fundamentals like arrays, functions, and randomization, showcasing how a basic script can still deliver a creative and entertaining experience.

Mindful Leaf

Mindful Leaf is a calming, wellness-focused web project designed to promote natural remedies and holistic self-care. The site combines clean, earthy design elements with user-friendly navigation to guide visitors through tips on plant-based healing, DIY herbal treatments, and mindful living practices. From aloe vera’s soothing powers to thyme’s antimicrobial benefits, Mindful Leaf educates users on everyday herbs and how to use them effectively. Built with accessibility and simplicity in mind, the site reflects a gentle aesthetic that mirrors the peaceful, intentional lifestyle it encourages.
